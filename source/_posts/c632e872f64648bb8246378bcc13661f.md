## 패키지 정의(Package Definition)

블록을 논리적으로 묶어서 패키지로 관리합니다. 패키지 내에 포함된 블록들이 모듈화되어 관리될 수 있습니다.

### 표

| Package Name | Description       | Contained Classes    |
| ------------ | ----------------- | -------------------- |
| DrawingApp   | 그림판 소프트웨어의 주요 패키지 | - DrawingApplication |
|              |                   | - Shape              |
|              |                   | - Color              |
|              |                   | - Point              |

### 필드와 값

1. **Package Name (패키지 이름)**
   - 패키지의 이름. Required

2. **Operation (작업, 동작)**
   - 메서드 이름을 기입합니다.
